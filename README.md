## Overview

This To-Do List app allows users to add tasks and remove them easily.  
All interactions happen dynamically without page reloads, and tasks are saved in the browser using **LocalStorage**, so they persist across sessions.

---

## Features

- Add new tasks  
- Display tasks instantly  
- Delete individual tasks  
- Tasks saved using **LocalStorage** (persistent storage)  
- Simple and minimal user interface  
- No external libraries

## How To Run

1. Download or clone the repository  
2. Open the project folder  
3. Open `index.html` in your browser  

No setup or installation required.

---

## How It Works

- Type a task in the input field  
- Click **Add Task**  
- The task appears in the list and is **saved in LocalStorage**  
- Refreshing or reopening the page will still show saved tasks  
- Click **Delete** to remove any task (also removes it from LocalStorage)
